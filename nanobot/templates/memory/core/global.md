# 🌐 Core Global Knowledge 
> 本文档定义了世界的物理规律与基座事实，它是应对所有访客对话的终极底层约束。当下方两区内容发生冲突时，第一区（Master认定）拥有无法撼动的最高判定权。

### 👑 1. Master 认定的绝对真相 (Absolute Truths)
- （属于 Master 主理人的个人意志与特许公域口径。客体共识如果与其冲突，必须服从本区真理。）

### 👥 2. 客体/群体共识总结的真相 (Consensus Facts)
- 来自于日常访谈提炼后确认的公共客观事实（如某种技术的常识、某个公司的客观新闻）。
- （该区权限较低，仅在不违背第一区时作为普世知识元应用。）
